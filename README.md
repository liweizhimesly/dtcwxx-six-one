# 芙蓉区大同崇文小学六年一班班级网站

这是一个由学生自主创建和维护的六年级一班非官方班级网站，用于记录班级生活、分享创意和资源。

## 🏫 网站简介
- **网址**：[https://liweizhimesly.github.io/dtcwxx-six-one/](https://liweizhimesly.github.io/dtcwxx-six-one/)
- **性质**：非官方、学生个人创建
- **目的**：班级记忆存档、同学交流、创意分享

## 📁 网站内容
- **班级论坛**：记录同学们的日常发言和趣事
- **纸质游戏库**：展示班级同学自创的课间游戏
- **资源分享**：分享学习工具和同学自研的编程项目（如Iazylang解释器）

## 🛠 技术栈
- 纯静态网站 (HTML/CSS/JavaScript)
- 托管于 GitHub Pages

## 🌟 特色
1. **完全由学生开发**：从设计到代码均由班级同学完成
2. **真实班级记忆**：论坛内容源自真实的同学互动
3. **创意展示**：包含自创游戏和编程项目

## 📄 协议
本网站内容仅供班级内部交流使用，部分资源遵循其各自的开源协议。
